Compute phase-amplitude coupling¶

tensorpac:

Pac([idpac, f_pha, f_amp, dcomplex, cycle, …])

Compute Phase-Amplitude Coupling (PAC).

EventRelatedPac([f_pha, f_amp, dcomplex, …])

Compute the Event Related Phase-Amplitude Coupling (ERPAC).

PreferredPhase([f_pha, f_amp, dcomplex, …])

Compute the Preferred Phase (PP).

Individual methods¶

tensorpac.methods:

PAC methods¶

If you don’t want to use the tensorpac.Pac class, you can also manually import the method of your choice and use it on phase / amplitude to compute PAC.

mvl(pha, amp)

Mean Vector Length.

kld(pha, amp[, n_bins])

Kullback Leibler Distance.

hr(pha, amp[, n_bins])

Heights ratio.

ndpac(pha, amp[, p])

Normalized direct Pac.

plv(pha, pha_amp)

Phase Locking-Value.

gcpac(pha, amp)

Gaussian Copula Phase-amplitude coupling.

Event Related PAC methods¶

erpac(pha, amp)

Correlation coefficient between a circular and a linear random variable.

ergcpac(pha, amp[, smooth, n_jobs])

Event Related PAC computed using the Gaussian Copula Mutual Information.

Preferred phase¶

preferred_phase(pha, amp[, n_bins])

Compute the preferred phase.

Surrogates methods¶

swap_pha_amp(pha, amp)

Swap phase / amplitude trials.

swap_blocks(pha, amp)

Swap amplitudes time blocks.

time_lag(pha, amp)

Introduce a time lag on phase series.

Normalize the phase amplitude coupling.
