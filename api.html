tensorpac.methods:

PAC methods¶

If you don’t want to use the tensorpac.Pac class, you can also manually import the method of your choice and use it on phase / amplitude to compute PAC.

mvl(pha, amp)

Mean Vector Length.

kld(pha, amp[, n_bins])

Kullback Leibler Distance.

hr(pha, amp[, n_bins])

Heights ratio.

ndpac(pha, amp[, p])

Normalized direct Pac.

ps(pha, amp)

Phase Synchrony (Penny, 2008; Cohen, 2008).

gcpac(pha, amp)

Gaussian Copula Phase-amplitude coupling.

Event Related PAC methods¶

erpac(pha, amp)

Correlation coefficient between a circular and a linear random variable.

ergcpac(pha, amp[, smooth, n_jobs])

Event Related PAC computed using the Gaussian Copula Mutual Information.

Surrogates methods¶

Swap amplitudes time blocks.

time_lag(pha, amp)

Introduce a time lag on phase series.
